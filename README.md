## Installation

To set up this project locally, follow these steps:

1. **Clone the repository**

2. **Install dependencies**

   ```sh
   npm install
   ```

3. **Start the development server**

   ```sh
   npm start
   ```

   The application should now be running on `http://localhost:3000`.
